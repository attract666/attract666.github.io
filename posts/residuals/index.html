现代的残差连接

在现代LLM中有两种Norm的形式，分别是Pre-Norm和Post-Norm，Pre-Norm是指 $x_{n+1} = x_{n} + f(norm(x_{n}))$，而Post-norm是指 $x_{n+1} = norm(x_{n} + f(x_{n}))$。后者是标准Transformer中所展现的，保证了主干方差的恒定，每层对x都可能有较大的影响，代价是没有从头到尾的恒等路径，梯度难以控制，因此会认为更难收敛，但finetune效果会更好一点。而前者则是现代LLM主流的方案，由于 $f(norm(x_{n}))$ 的方差是不随层数变化的，于是x的方差会在主干上随层数积累，到了深层后，单层对主干的影响可以视为小量，所以会有 $x_{n+2} = x_{n+1} + f(norm(x_{n+1})) \approx x_{n} + 2f(norm(x_{n}))$，可以看成是在模型宽度上的拓展，所以相对好训练。因此其实两者之争，更多的是残差流的内斗。

标准Residuals

标准的residuals，大家其实应该已经很熟悉了，$x_{n+1} = x_{n} + f(x_{n})$，其中 $x_{n}$ 希望把梯度从顶层传递到底层，所有层都在通过主干通道向维度$d$中叠加信号，浅层的语义和深层的语义都叠加在$d$维空间中。这显然是有问题的，同时每层都是在以1:1的比例读写主干道，单层结构如下图：

让我们从第t层输出$y_{t} = f_{t}(x_{t-1})$的角度，来看看标准的residuals： $$x_{t} = x_{t-1} + y_{t} = x_{0} + … + y_{t}$$ 其中$x_{0}=y_{0}$，也就是说标准residual做的就是将每一层的输出使用相等的权重求和，可以形式化为：

$$y_{t+1} = f_{t+1}(x_{t}) = f_{t+1}(\sum^{t}_{i=0}a_{i}y_{i})$$

HC

因此HC(Hyper-Connections)的想法很直接，把主干道从一条$d$维扩成$n_{hc} * d$维，形式上： $$x_{n+1} = H_{n+1}^{res}x_{n} + H_{n+1}^{post}f(H_{n+1}^{pre}x_{n})$$

单层结构如下图：

其中$H_{n}^{res} \in R^{n \times n}$、$H_{n}^{pre} \in R^{1 \times n}$、$H_{n}^{post} \in R^{n \times 1}$，让我们依旧从第t层输出$y_{t}$的角度来看看HC： $$x_{t} = H_{t}^{res}x_{t-1} + H_{t}^{post}y_{t} = H_{t}^{res}H_{t-1}^{res}x_{t-2} + H_{t}^{res}H_{t-1}^{pos}y_{t-1} + H_{t}^{post}y_{t}$$ 那么： $$y_{t+1} = f_{t+1}(H_{t+1}^{pre}x_{t}) = f_{t+1}(\sum^{t}_{i=0} H_{t+1}^{pre}\Phi(t,i)H_{i}^{post}y_{i})$$

其中 $$\Phi(t, i) = \begin{cases} I & \text{if } t = i \\ H_t^{res} H_{t-1}^{res} \dots H_{i+1}^{res} & \text{if } t > i \end{cases}$$

而$H_{t+1}^{pre}\Phi(t,i)H_{i}^{post}$是一个标量，也符合上面的形式化定义，其中三个$H$因为具体实现时使用带tanh激活的线性层连乘存在爆炸的风险，同时也无法保证$a_{i}$非负（如果$a_{i}$有正有负说明某些层正向优化，有的层负向优化，这对模型学习不够友好）

mHC

而mHC则就是对HC的改进，来解决上面两个问题，下面给出了mHC的单层结构图：

首先是非负的问题，因此mHC将tanh激活换成了sigmoid，但是如何控制前后向爆炸的问题呢？这就需要引出mHC的主角：双随机矩阵

让我们先来看看谱范数，反映了线性变换对向量的最大伸缩幅度。而对于任意一个矩阵都可以做奇异值分解，奇异值的最大值就是这个矩阵的谱范数，如果谱范数小于等于1，那么意味着它作用于其他矩阵都不会让它变大

双随机矩阵是指行之和为1，列之和也为1，且所有元素都大于0的矩阵。双随机矩阵的谱范数小于等于1，且在乘法下封闭，从而保证了使用双随机矩阵可以保证数值稳定。

可是我们的$H^{res}$是学出来的，那么要怎么保证它在双随机矩阵的流形上呢？

先随机生成一个$H^{res}$，之后通过Sinkhorn-Knopp算法将$H^{res}$变成离它最近的双随机矩阵，再用新的$H^{res}$去做残差更新。而Sinkhorn-Knopp就是对矩阵行和列反复进行归一化
